2. 博客展示截图请务必使用`https`开头的链接，可使用[sm.ms图床](https://sm.ms/)
3. 只要你的博客为`Hexo`，且文章多于3篇即可进入本站展示

格式如下: 
```json
{
name: '名字', 
desc: '描述',
link: '链接', 
themeimg: '博客展示截图' 
}
```
