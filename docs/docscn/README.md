## 常用说明
:::tip
本站的多数文档凡涉及复杂命令的，都会为你整理出常用命令，你可以只去学习常用命令。记住多余冗杂的命令参数，其实是毫无用处的。当然，当你可以不看文档便能自己搭建且配置完整的博客，你可以此时再去了解其他命令或者参数
:::
